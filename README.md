# vite-redux-boilerplate

This boilerplate includes the following:

- Vite
- React
- Redux Toolkit
- RTK Query
- tailwindcss
- eslint/prettier
- jest/TS/react-testing-library/MSW
- GitHub Actions CI.

# Installation

```
npx degit rakshithp7/vite-redux-boilerplate myapp
```

```
cd myapp
yarn install
yarn dev
```

or npm

```
cd myapp
npm install
npm run dev
```
